# 98.验证二叉搜索树

### 题目

【middle】给你一个二叉树的根节点 root ，判断其是否是一个有效的二叉搜索树。

有效 二叉搜索树定义如下：

- 节点的左子树只包含 小于 当前节点的数。
- 节点的右子树只包含 大于 当前节点的数。
- 所有左子树和右子树自身必须也是二叉搜索树。

[https://leetcode.cn/problems/validate-binary-search-tree/](https://leetcode.cn/problems/validate-binary-search-tree/%E2%80%B8)

### 方法

【中序遍历】

回顾二叉搜索树的性质：

- 二叉搜索树中序遍历的结果为有序序列
- 中序遍历结果为有序序列的二叉树 一定是 二叉搜索树

【时间复杂度】O(n)

【空间复杂度】O(n)

### 代码

```cpp
// 中序遍历，搜索二叉树中序遍历必然有序，反之也成立
class Solution {
private:
    vector<int> res;
public:
    void search(TreeNode* root) {
        if (!root)
            return;
        search(root->left);
        res.emplace_back(root->val);
        search(root->right);
    }
    bool isValidBST(TreeNode* root) {
        search(root);
	// 对中序遍历的结果进行判断
        int n = res.size();
        for (int i = 0; i < n - 1; i++)
            if (res[i] >= res[i + 1])
                return false;
        return true;
    }
};
```
