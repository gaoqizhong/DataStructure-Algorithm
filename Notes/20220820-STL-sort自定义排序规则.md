# CppSTL sort

1、基本用法

```cpp
sort(v.begin(), v.end()); 					// 默认升序
sort(v.begin(), v.end(), greater<int>()); 	// 降序
```

2、自定义排序规则（重载法）

```cpp
// 重载()
struct myRule {
	bool operator() (const int& a, const int& b) {
		return a > b; 
	}
};
sort(v.begin(), v.end(), myRule());
```

3、自定义排序规则（定义函数）

```cpp
bool myRule(const int& a, const int& b) {
    return a > b;
}
sort(v.begin(), v.end(), myRule);
```
